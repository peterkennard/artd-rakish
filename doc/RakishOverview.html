== Package Overview Rakish is a build system package extending the capability of the ::Rake depedency based task management system. Rakish extends the capabilitiues of ::Rake with build module configuration and management features and a host of useful utilites for creating tasks and a facile ability for writing and including custom modules for building different types of output. Several types of modules are included here in the base ::Rakish package for building Java, Jar, Zip, and Apache Tomact War file outputs.
